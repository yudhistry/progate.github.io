HTML & CSS

HTML merupakan bahasa penanda yang digunakan untuk memberikan konten (teks) struktur dan makna semantik. Dan CSS adalah bahasa yang digunakan untuk memberikan konten gaya penampilan yang bagus.

jQuery

jQuery adalah library JavaScript yang populer. Bahasa pemrograman ini dibuat oleh John Resig, tepatnya pada tahun 2006, untuk memudahkan para developer dalam menggunakan dan menerapkan JavaScript di website.

Ruby

Ruby adalah bahasa pemrograman dinamis berbasis skrip yang berorientasi objek. Tujuan dari ruby adalah menggabungkan kelebihan dari semua bahasa-bahasa pemrograman skrip yang ada di dunia. Ruby ditulis dengan bahasa pemrograman C dengan kemampuan dasar seperti Perl dan Python.

PHP

PHP adalah bahasa pemrograman yang sering disisipkan ke dalam HTML. PHP sendiri berasal dari kata Hypertext Preprocessor.
